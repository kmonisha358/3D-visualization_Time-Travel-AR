With Time Travel AR, users may visually explore 15 famous landmarks spread across 15 Indian states, each of which represents a distinct period of history and culture. The immersive 3D Augmented Reality experience was created using Unity (URP).

With the use of Unity, C# scripting, Blender, and Sketchfab models, this project blends contemporary technology with heritage visualisation to create an engaging learning experience through interactive 3D models, augmented reality time travel, and educational hints.

# Overview:

Through the creation of a virtual time-travel experience, the Time Travel AR project seeks to unite history and technology. Visitors can visit each Indian state's historical monument, observe it in three dimensions, and engage with it through tags, animations, and voice narration by scanning augmented reality markers or navigating the app.

Constructed using Unity's Universal Render Pipeline (URP), it provides excellent lighting, shadows, and environmental depth, resulting in a lifelike, realistic, and instructive experience.
